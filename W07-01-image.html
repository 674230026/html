1) Image Basics src/alt

รูปภาพใน HTML ใช้แท็ก imgโดยต้องมีsrcและควรมีaltเสมอ

            <img src="images/photo.jpg" alt="ภาพตัวอย่าง">

📢 alt สำคัญมาก: ช่วย SEO + ใช้แทนภาพเมื่อโหลดไม่ได้ + ช่วยผู้ใช้ Screen Reader

​

2. Resize & Aspect Ratio object-fit

ถ้ากำหนดทั้งความกว้างและสูง ให้ใช้ object-fit เพื่อไม่ให้ภาพบิด

/* ครอปภาพให้พอดีกรอบ */
.img-fixed {
 width: 260px;
 height: 160px;
object-fit: cover;
}

cover = ✨ครอปให้เต็มกรอบ, contain = เห็นภาพครบ (อาจมีขอบว่าง)

3) Original Fitcontain fill cover

ตัวอย่างการใช้ object-fit แบบต่างๆ

.obj-fit-image {
    width: 800px;
    height: 200px;
    border: 1px solid black; 
}

.obj-fit-image.contain { object-fit: contain; }
.obj-fit-image.contain { object-fit: fill; }
.obj-fit-image.contain { object-fit: cover; }

ภาพต้นฉบับ

object-fit: contain; ➡️ เห็นภาพครบรักษาสัดส่วน (อาจมีขอบว่าง)

object-fit: fill; ➡️ ยึดภาพให้เต็มกล่อง ภาพอาจบิด (สัดส่วนเพี้ยน)

object-fit: cover; ➡️ ครอบภาพให้เต็มกล่อง รักษาสัดส่วน บางส่วนของภาพจะถูกตัดออก

4) Image Link <a> + <img>

คลิกที่รูปเพื่อเปิดลิงก์ เหมาะกับ banner / thumbnail

        <a href="https://www.youtube.com" target="_blank">
        <img src="images/banner.jpg" alt="Banner">
        </a >

ใช้target="_blank" เมื่ออยากเปิดในแท็บใหม

5) Image Gallery grid grid

แสดงรูปหลายรูปด้วย Grid ให้จัดเรียงสวยและเป็นระเบียบ

.gallery { display: grid; grid-template-columns: repeat(3, 1fr); gap: 12px; }
    .gallery img { height: 120px; object-fit: cover; }

📌ถ้ารูปขนาดไม่เท่ากัน ใช้object-fit: cover;จะช่วยให้ดูสมํ่าเสมอ

6) Responsive Images max-width:100%

ทพให้ภาพไม่ล้นการ์ดและย่อขยายตามหน้าจอ 

img { max-width: 100%; height: auto; }

💡เทคนิคนี้เป็นพื้นฐานสำคัญของเว็บ responsive

7) Figure + Caption งานนักศึกษา

<figure>
            <img src="..." alt="...">
            <figcaption>รูปที่ 1: คำอธิบายภาพ</figcaption>
            </figure>

ให้นักศึกษาทำหน้า"My Favorite Place/Product/Technology"โดยมีภาพอย่างน้อย 5 ภาพ และต้องมี 1) รูปแบบ Gallery 2) รูปแบบ image Link 3) รูปพร้อม Caption

8) ใช้รูปจากเครื่อง (Local Images) VS Code

จัดโฟลเดอร์รูปให้เป็นระเบียบ เช่น images/ และอ้างอิงแบบ relative path

my_project/
 ├── index.html
 └── images/
      ├── photo1.jpg
      └── banner.png
<img src="images/photo1.jpg" alt="My Photo">

แนะนำให้ตั้งชื่อไฟล์รูปเป็นภาษาอังกฤษ ไม่มีช่องว่าง เช่น my-photo-1.jpg
